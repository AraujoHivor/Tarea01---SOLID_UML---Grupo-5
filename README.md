# Sistema PetDayCare

**PetDayCare** es un sistema diseñado para la **gestión de reservas y servicios especializados para el cuidado temporal de mascotas**.  
El proyecto aplica principios de **UML** y los **principios SOLID**, integrando casos de uso, diagramas de clases, diagramas de secuencia y su respectiva implementación en Java.

---

## Descripción general

El sistema permite a los usuarios:
- Registrar sus mascotas y sus características (especie, raza, edad, tamaño, necesidades especiales).  
- Reservar servicios como hospedaje, guardería, paseos, entrenamiento o bienestar.  
- Verificar disponibilidad y confirmar pagos en línea.  
- Recibir notificaciones y reportes sobre el estado de las reservas.  
- Reportar incidentes y gestionar su resolución mediante el administrador del centro.

Los **administradores de centros** pueden configurar servicios, precios, horarios y gestionar incidentes o comentarios de los cuidadores.

---

## Casos de uso implementados

1. **Registrar Mascota**  
2. **Reservar Servicio**  
3. **Confirmar Pago**  
4. **Atender Incidente**

Cada caso cuenta con:
- Diagrama de secuencia  
- Flujo principal y alternativos  
- Clases y métodos asociados
